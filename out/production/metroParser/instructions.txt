файл ALL_PRODUCTS.xlsx содержит все продукты.

Для импорта информации о товарах на сайт необходимо:
	1. экспортировать файл с товарами.
	2. Дополнить его информацией о новых товарах, сохранить.
	3. Импортировать дополненный файл.

Экспорт файла:
	В левой панели заходим: Система->Обсулживание->Экспорт / Импорт.

Добавление информации в файл импорта:
	1.копировать значения из столбца "Название" файла ALL_PRODUCTS.xlsx в столбцы name(en-gb) и name(ru-ru)
	2.инкрементировать значения столбца product_id для каждого добавленного товара.
	3.копировать соответствующие значения цены из файла ALL_PRODUCTS в файл импорта (колонка price, лист Products)
	4.Копировать соответствующие значения описания из файла ALL_PRODUCTS в файл импорта (колонки description(en-gb)	description(ru-ru))
	5.Ссылка на картинки. Все картинки в папке /image/catalog/images/categories/parsedimages/.
	6.Заполнить значения других столбцов по необходимости.
	7.Состав по необходимости можно копировать либо в атрибуты, либо конкатенировать со строкой-описанием и т.д.
	8.Для того чтобы импортировать таблицы(Колонки "общая информация" и "Пищевая ценность" файла ALL_PRODUCTS.xlsx) Необходимо экспортировать
		 с сайта файл с атрибутами, затем создать в нем необходимые группы атрибутов и названия атрибутов. (Общая информация и Пищевая ценность уже частично добавлены.),
		Сохранить изменения и импортировать файл с атрибутами.
		После этого в файле импорта продуктов на листе ProductAttributes заполняем значения соответствующих атрибутов.

Сохраняем файл импорта продуктов, импортируем.
